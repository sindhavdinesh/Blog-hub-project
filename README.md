## рҹ”§ Installation & Local Setup

### Step 1 вҖ” Clone the Repository

```bash
git clone https://github.com/sindhavdinesh/Blog-hub-project.git
cd Blog-hub-project
```

### Step 2 вҖ” Install Dependencies

```bash
npm install
```

### Step 3 вҖ” Configure Environment Variables

Create a `.env` file in the root:

```env
# в”Җв”Җ Server в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
PORT=5000

# в”Җв”Җ MongoDB Atlas в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
MONGODB_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/blogdb?retryWrites=true&w=majority

# в”Җв”Җ Cloudinary в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

# в”Җв”Җ Frontend в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

### Step 4 вҖ” Launch Dev Environment

```bash
npm run dev
```
